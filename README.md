OneSpot Finance

A Unified Personal Finance & Investment Management Platform

📌 Project Overview

OneSpot Finance is a full-stack web application designed to help users manage all their financial assets in one place.

The platform allows users to:

Track income and expenses

Set and monitor financial goals

Manage investments

Monitor Indian equity holdings

View financial summaries through a dashboard

⚠️ This project is currently under active development. Some features are incomplete and improvements are ongoing.

🔥 Core Features & Functionalities
👤 1. Secure Authentication System

User Signup & Login

Password Hashing (secured using backend encryption)

JWT Token-based authentication

Protected routes (users can only access their own data)

This ensures each user has a secure and private financial space.

📊 2. Dashboard Overview

The dashboard provides:

Total Income

Total Expenses

Total Investments

Goal Progress Overview

Asset Summary Snapshot

Users can quickly understand their financial status in one screen.

💸 3. Expense & Income Management

Users can:

Add income entries

Add expense records

Categorize transactions

View updated totals dynamically

This helps track spending habits and savings patterns.

🎯 4. Financial Goal Management

Users can:

Create short-term goals

Create long-term goals

Set target amount

Set target date

Track progress toward completion

This helps in structured financial planning.

📈 5. Investment Tracking

Currently supports:

Manual entry of investments

Categorization of investment types

Tracking invested amount

🇮🇳 6. Indian Equity Tracking (Work in Progress)

Users can:

Add Indian equity stocks manually

Track invested value

Auto-calculate portfolio value

🪙 7. Crypto Tracking (Planned Feature)

🚧 Under Development

Planned features:

Add crypto holdings

Fetch live crypto prices

Show real-time portfolio value

Market feed integration

💡 How Users Can Manage Assets in One Place

OneSpot Finance centralizes:

Income records

Expenses

Goals

Equity investments

(Upcoming) Crypto assets

Instead of switching between multiple apps, users can:

Monitor financial growth

Track investment diversification

Plan long-term goals

Analyze asset allocation

All from a single dashboard interface.
